문서 제목 Document Title

1. 서론

이것은 A4 규격으로 출력 가능한 HTML 문서 템플릿입니다. 이 템플릿은 웹 브라우저에서 보기에도 좋고, PDF로 출력했을 때도 정확한 A4 크기로 인쇄됩니다.

각 페이지는 210mm × 297mm의 A4 표준 크기를 가지며, 적절한 여백과 함께 가독성을 고려한 레이아웃으로 구성되어 있습니다.

1.1 주요 특징

  • A4 규격 강제 출력 (@page size: A4)
  • 자동 페이지 번호
  • 헤더와 푸터 지원
  • 프린트 최적화
  • 반응형 텍스트 레이아웃

1.2 사용 예시 테이블

항목 설명 비고
페이지 크기 210mm × 297mm (A4 표준) CSS로 강제 지정
여백 상하좌우 20mm 가독성 최적화
폰트 Noto Sans KR / Inter 웹폰트 적용
문서 제목 Document Title

2. 사용 방법

2.1 기본 구조

새로운 페이지를 추가하려면 <div class="page"> 요소를 복사하여 사용하시면 됩니다. 각 페이지는 자동으로 새로운 페이지로 분할됩니다.

참고사항:

이 템플릿은 Chrome, Firefox, Safari 등 모든 주요 브라우저에서 정상적으로 작동하며, "인쇄" 기능을 통해 PDF로 저장할 수 있습니다.

2.2 스타일 커스터마이징

  1. 색상 변경: CSS 변수 또는 직접 색상값 수정
  2. 폰트 변경: Google Fonts 링크 및 font-family 수정
  3. 여백 조정: .page 클래스의 padding 값 수정
  4. 헤더/푸터 커스터마이징: 해당 클래스 스타일 수정
